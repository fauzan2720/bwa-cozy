# Cozy

Aplikasi pencarian kos-kosan online yang dibuat menggunakan Flutter SDK dalam kelas online di Buildwith Angga (BWA).

![thumbnail](https://user-images.githubusercontent.com/74108522/213167427-3fc45edd-3266-409e-b072-a052e8816838.jpg)

## Mockup

![mockup](https://user-images.githubusercontent.com/74108522/215240897-763a1041-dbe9-44e1-99b4-f5509910233a.png)

## Features

1. Menerapkan arsitektur MVC
2. Integrasi API
3. Fitur Wishlist dengan package hive sebagai local storage
4. Menerapkan useMaterial3
5. Menggunakan URL Launcher untuk direct ke Google Maps dan Telepon

## Implementation

1. MVC Architecture
2. State Management Provider
3. Consume API
4. Local Storage Hive
5. Deploy to Google Play Store
